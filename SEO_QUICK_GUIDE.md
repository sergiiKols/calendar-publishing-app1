## 🎯 Основной рабочий процесс

### Сценарий 1: Быстрое добавление ключевых слов
```
1. Открыть /seo
2. Нажать "Добавить ключевые слова" (синяя кнопка)
3. Выбрать проект из списка
4. Выбрать язык и регион
5. Ввести ключевые слова (по одному на строку)
6. Нажать "Добавить"
7. Подождать обработки
8. Нажать 👁️ для просмотра результатов
```

### Сценарий 2: Сбор семантического ядра
```
1. Открыть /seo
2. Нажать "Собрать семядро" (фиолетовая кнопка)
3. Ввести 3-5 seed-ключевых слов
4. Система автоматически:
   - Найдёт связанные ключевые слова
   - Получит метрики (SV, CPC, KD)
   - Сгруппирует по семантике
   - Определит интенты
5. Результат появится в секции "Семантические кластеры"
6. Нажать ℹ️ для детального просмотра
7. Экспортировать CSV при необходимости
```

### Сценарий 3: Управление проектами
```
1. Открыть /seo
2. Нажать "Wizard" (обведённая синим кнопка)
3. На шаге 1 увидите все проекты
4. Для редактирования:
   - Нажать "Редактировать" на карточке
   - Изменить данные
   - Нажать "Сохранить"
5. Для удаления:
   - Нажать "Удалить"
   - Подтвердить второй раз
```

## 💡 Подсказки

### Иконки и их значение:
- 👁️ = Просмотр результатов
- ℹ️ = Детальная информация
- 📥 = Скачать CSV
- ✏️ = Редактировать
- 🗑️ = Удалить
- 🔄 = Обновить
- ✨ = Семантическое ядро (кластеризация)

### Статусы ключевых слов:
- ⏳ Ожидание (pending) - в очереди
- 🔄 Обработка (processing) - идёт запрос к API
- ✅ Завершено (completed) - данные получены
- ❌ Ошибка (failed) - что-то пошло не так

### Типы интентов:
- 🔵 informational - информационный
- 🟢 transactional - транзакционный
- 🟡 commercial - коммерческий
- 🟣 navigational - навигационный
- ⚫ local - локальный

## 🆘 Частые вопросы

**Q: Можно ли добавлять ключевые слова к проекту после его создания?**
A: Да! Используйте кнопку "Добавить ключевые слова" на главной странице SEO.

**Q: Как посмотреть подробные данные по ключевому слову?**
A: Нажмите кнопку 👁️ в таблице напротив нужного ключевого слова.

**Q: Где хранятся результаты парсинга?**
A: В таблицах БД: `seo_results`, `seo_serp_positions`, `seo_cluster_keywords`.

**Q: Можно ли экспортировать данные?**
A: Да! В детальном просмотре кластера есть кнопка "Экспорт CSV".

**Q: Сколько ключевых слов можно добавить за раз?**
A: До 100 ключевых слов в одном запросе.
